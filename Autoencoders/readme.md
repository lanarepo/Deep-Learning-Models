## **Autoencoders (Feature Learning)** <br>
Autoencoders are unsupervised neural networks that learn to compress data into a lower-dimensional representation (encoding) and then reconstruct it back to the original format (decoding).

🔵 **The Architecture** <br>
**Encoder**: Compresses the input into a "Latent Space" (Bottleneck). <br>
**Bottleneck**: The compressed layer containing the most vital features. <br>
**Decoder**: Reconstructs the input from the latent representation.

🔵 **Learning Goals** <br>
<ul>
  <li>Understand Dimensionality Reduction.</li>
  <li>Learn about Reconstruction Loss (MSE).</li>
  <li>Explore applications like Image Denoising and Anomaly Detection.</li>
</ul>
